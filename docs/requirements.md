# RoboKing Sales Platform CRM Requirements

## Roles

1. Owner
2. Manager
3. PA/Admin Assistant
4. Employee

Owner and Manager have full access to all accounts/data. Employees see own/assigned leads only. PA coordinates reminders and employee follow-ups.

## Lead rules

- Employees can manually create leads.
- All employees can import Excel leads.
- Owner/Manager can see every lead immediately and reassign anytime.
- Only Owner/Manager can delete imported data/leads.
- Exact duplicate phone/email is blocked by default.
- Owner/Manager can override duplicate block.
- Duplicate warning must show clearly: “Duplicate contact found”.

## Communication

- Call client using click-to-call.
- Employee manually enters call details after conversation.
- Call recording upload is mandatory.
- Call recording supports common audio/video formats.
- WhatsApp opens employee’s own WhatsApp app/Web with client number/name and prefilled admin-controlled template.
- Email compose modal inside CRM.
- Email sends from employee’s own Hostinger email via SMTP.
- Inbox replies sync back into CRM via IMAP.
- Every sent email auto-BCCs admin/founder email.
- Attachments are allowed for everyone unless restricted later.

## Internal chat

- Admin can create direct/group chats.
- Employee-to-employee chat only if admin allows.
- Admin can read all chat history.
- Chat can be linked to lead/deal.

## Reports

- Team performance
- Lead source
- Follow-up pending
- Conversion
- Revenue pipeline
- Employee activity
- Excel import
- Lost lead
